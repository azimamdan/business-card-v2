## Lessons Learned
- **Architecture:** `JSONB` for block data significantly accelerated development of new block types without database migrations.
- **Branding:** Moving branding and theming to Phase 6 allowed for core functionality to stabilize first, but required a major grep/replace effort for colors (solved with semantic CSS variables).
- **UX:** The side-by-side live preview in the dashboard is the standout feature for user retention.
- **Efficiency:** Supabase's unified auth/storage/database layer reduced boilerplate by ~30% compared to fragmented stacks.
